# Estruturas de Roteiro

Estruturas testadas para escrita de roteiros virais. Cada uma serve para um tipo diferente de video.

## Estruturas disponiveis

| Estrutura | Etapas | Melhor para |
|---|---|---|
| [AIDA](aida.md) | Atencao > Interesse > Desejo > Acao | Videos de venda, conteudo educativo com CTA |
| [IHC](ihc.md) | Identificacao > Historia > Conteudo | Conexao emocional, plantar ideias sem parecer venda |
| [Jeito Errado x Jeito Certo](jeito-errado-jeito-certo.md) | Conflito > Erro > Virada > Acerto > Mudanca | Tutoriais, posicionamento de autoridade, contraste |
| [Ele, Eu, Voce, Futuro](ele-eu-voce-futuro.md) | Ele > Eu > Voce > Futuro | Storytelling compartilhado, narrativa que cresce |
| [Analise](analise.md) | Conflito > Resultados > 3 Motivos > Moral | Surfar tendencias, analisar criadores, posicionar metodo |

## Como os agentes usam estas estruturas

- O **roteirista** sugere a estrutura mais adequada ao tema e objetivo do usuario
- O **analista-de-principios** identifica qual estrutura um video analisado esta usando
- O **critico** verifica se a estrutura escolhida foi aplicada corretamente
